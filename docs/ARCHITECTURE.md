# Architecture Overview

The portfolio combines four complementary systems:

1. Retrieval-backed fraud investigation (Project 1)
2. Real-time supervised fraud scoring service (Project 2)
3. Multi-agent monitoring and SAR generation (Project 3)
4. Fine-tuned narrative classifier for transaction text (Project 4)

Shared concerns across all projects:
- FastAPI-based service interfaces
- Containerized deployment with Docker
- Reproducible tests and metrics
- Cloud-ready deployment patterns
