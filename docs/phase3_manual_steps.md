# Phase 3 Manual Steps (Cannot Be Fully Automated from Terminal)

## Profile Updates
- Update Indeed profile (experience, skills, education, certifications, preferences).
- Update LinkedIn headline, about, featured links, projects, and skills.
- Send 3 LinkedIn outreach messages (Kata.ai, Kredivo, Devoteam).

## Job Applications
- Submit the top-10 applications via Indeed/LinkedIn/Glints/Company pages.
- Track responses in `docs/application_tracker.csv`.

## Media Capture
- Record 4 demo GIFs and capture full-page screenshots.
- Add GIF assets to project repos (`assets/demo.gif`).
- Optional: pin 4 repos on GitHub profile manually.
